# Product Overview

Excuse Killer is a gamified React web application designed to help users overcome procrastination through challenge tracking, streak maintenance, and achievement unlocking.

## Core Features

- **Challenge Management**: Users create challenges with three intensity levels (chill/10min, normal/20min, hardcore/30min) and target dates
- **Timer System**: Built-in circular progress timer for focused work sessions
- **Progress Tracking**: Daily streaks, weekly/monthly stats, and completion history
- **Achievement System**: Unlockable badges based on completions and streaks
- **Glass Morphism UI**: Dark-themed interface with modern glass effects
- **Local Storage**: Browser-based persistence, no backend required

## Game Mechanics

- **XP System**: Chill (30 XP), Normal (50 XP), Hardcore (75 XP)
- **Streak Logic**: Consecutive days with completions, must include today
- **Title Badges**: Starter (0-2), Overcomer (3-6), Super Overcomer (7-14), Unstoppable (15+)
- **Achievements**: First Step, Week Warrior, Unstoppable, Speed Demon, Consistency King, AI Believer

## Target Users

Individuals looking to build consistent habits, overcome procrastination, and track personal goals through gamification.
